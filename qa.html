FAQs

What kind of AI/ML projects have you worked on?

I have worked on AI-based systems including chatbot development, Retrieval-Augmented Generation (RAG) pipelines, and machine learning models for real-world applications such as language processing and intelligent assistants.

What is a RAG system?

Retrieval-Augmented Generation (RAG) is an approach that combines vector-based information retrieval with Large Language Models to generate accurate, context-aware responses using external knowledge sources.

How do you build AI chatbots?

I build AI chatbots using LLM APIs, prompt engineering, and context management techniques. I enhance chatbot performance by integrating RAG pipelines for better accuracy and reduced hallucinations.

What technologies and tools do you use?

I use Python, FastAPI, Pandas, NumPy, Scikit-learn, and LLM frameworks like LangChain. I also work with vector databases such as FAISS or Pinecone for building RAG systems.

Do you deploy AI models?

Yes, I deploy AI models and chatbot systems using FastAPI, creating APIs for real-time interaction and integration with frontend or external applications.

What is your approach to building AI systems?

My approach includes problem understanding, data preprocessing, model development, integrating LLMs or RAG pipelines, and deploying scalable APIs for real-world usage.
